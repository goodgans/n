# Инструкция для Markdown #

## Выделение текста ##

Чтобы выделить текст курсивом необходимо обрамить звездочками (*). Например: *Вот так*

Чтобы выделить текс полужирным, необходимо его обромить двойными звездочками (**). Например:   **вот так**
